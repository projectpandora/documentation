<PageDescription>
  API Tokens allow you to access our{' '}
  <a href="https://developers.projectpandora.xyz">public API</a> without a using
  username or password.
</PageDescription>

To create a new API Token:

1. Go to [profile settings](https://platform.projectpandora.xyz/profile) page and choose the **Tokens** tab.
2. Click **Create an API Token** button, input name and choose expires date for the new API Token.
3. Confirm by click **Add**. In the next section, you will be able to see your new token. Save this somewhere because you won't see it again.

To delete an API Token:

1. Go to [profile settings](https://platform.projectpandora.xyz/profile) page and choose the **Tokens** tab.
2. Click the **Delete** button next to the token name.
3. Press **Delete** in the confirm modal.
